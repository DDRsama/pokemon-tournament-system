# Pokemon Tournament System 3.3.0

## 中文

PTS 第一次实际应用后根据现场发现的问题修复了一些 bug。

### 新增与改进

- 增强后台误触保护：后台录入胜负和小分时增加确认，并禁止等待对手、BYE、TBD 对局被误操作。
- 完善 BO3 / BO5 支持：瑞士轮、小组赛、单败淘汰、双败淘汰会按阶段规则处理局分和胜局数。
- 完善 Top Cut 支持：修复 Top 4 / Top 16 不应套用旧 Top 8 结构的问题，Top 8 继续保留专用八强树 Overlay。
- 优化选手档案与参赛名：支持选手改名、参赛名与档案名分离，并同步已绑定的比赛记录。
- 优化直播房号流程：Top 4 / Top 8 淘汰赛同轮对局可共享直播房号，只有指定对局显示直播桌标签。
- 修复积分发放：非八强选手也会获得参赛分。
- 优化战报：清理旧版残留字段，补充阶段历史记录，修复部分英文阶段名漏出问题。
- 优化多语言：继续补齐中文、英文、日文界面文案和动态刷新。

### 字体与战报

- 移除不适合公开打包的私有字体资源。
- 改为内置可再分发的开源字体。
- PDF 战报现在由服务端嵌入项目内置的 Noto Sans SC / JP Medium 静态字体，不再依赖用户设备字体，减少移动端缺字和字体过细问题。
- 私有字体仍可通过 `/data/fonts` 自行挂载使用，授权由部署者自行确认。

## English

PTS 3.3.0 fixes bugs discovered after the first real-world tournament usage.

### Added And Improved

- Added stronger admin-side confirmation and safeguards for match result entry, including waiting-opponent, BYE, and TBD matches.
- Improved BO3 / BO5 handling across Swiss, groups, single elimination, and double elimination stages.
- Improved Top Cut handling so Top 4 and Top 16 no longer fall back to the old Top 8 structure, while Top 8 keeps the dedicated bracket overlay.
- Improved player profiles and tournament entry names, including player renaming and profile-bound record synchronization.
- Improved live-room handling for Top 4 / Top 8 playoff rounds, allowing the same live-room code to be shared while keeping only one match marked as the live table.
- Fixed participation points so players outside Top Cut can also receive participation points.
- Cleaned up exported reports, removed legacy fields, preserved stage history, and fixed leaked internal English stage labels.
- Continued Chinese, English, and Japanese i18n coverage improvements.

### Fonts And Reports

- Removed private font assets that should not be bundled in public releases.
- Switched bundled assets to redistributable open-source fonts.
- PDF reports now embed bundled static Noto Sans SC / JP Medium fonts on the server side, instead of depending on viewer-device fonts. This reduces missing glyphs and overly thin text, especially on mobile.
- Private fonts can still be mounted through `/data/fonts`; font licensing remains the deployer's responsibility.

## 日本語

PTS 3.3.0 は、初回の実運用後に現場で見つかった問題を修正するリリースです。

### 追加と改善

- 管理画面での勝敗・スコア入力に確認を追加し、対戦相手待ち、BYE、TBD の対戦が誤操作されないようにしました。
- スイスドロー、グループ、シングルエリミネーション、ダブルエリミネーションで BO3 / BO5 の処理を改善しました。
- Top 4 / Top 16 が旧 Top 8 構造に戻ってしまう問題を修正しました。Top 8 は専用トーナメント表 Overlay を維持します。
- プレイヤープロフィールと大会参加名を改善し、名前変更と紐付いた大会記録の同期に対応しました。
- Top 4 / Top 8 の決勝トーナメントで同一ラウンドの対戦に配信ルームコードを共有できるようにし、配信卓ラベルは指定対戦だけに表示します。
- Top Cut 以外のプレイヤーにも参加ポイントが付与されるよう修正しました。
- レポート出力を整理し、旧フィールドを削除し、ステージ履歴を保持し、一部の内部英語ラベルが表示される問題を修正しました。
- 中国語、英語、日本語の i18n 対応を継続して改善しました。

### フォントとレポート

- 公開リリースに含めるべきでない私有フォント素材を削除しました。
- 同梱フォントを再配布可能なオープンソースフォントへ切り替えました。
- PDF レポートは、閲覧端末のフォントに依存せず、サーバー側で同梱の静的 Noto Sans SC / JP Medium フォントを埋め込むようになりました。これにより、モバイル環境での文字欠けや細すぎる表示を軽減します。
- 私有フォントは引き続き `/data/fonts` から利用できます。フォントライセンスの確認はデプロイする利用者の責任です。
