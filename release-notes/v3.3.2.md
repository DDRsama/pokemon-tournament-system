# Pokemon Tournament System 3.3.2

## 中文

- 修复部分动态界面文案切换语言后仍残留中文的问题。
- PDF 战报现在会按目标语言生成文案，并优先使用 `/data/fonts/zh`、`/data/fonts/en`、`/data/fonts/ja` 中对应语言的字体。
- 修正 Swiss + Top Cut、小组赛和纯淘汰赛的最终名次显示，非八强选手不再显示为未排名。

## English

- Fixed dynamic UI text that could remain in Chinese after switching languages.
- PDF reports now render text in the selected target language and prefer the matching `/data/fonts/zh`, `/data/fonts/en`, or `/data/fonts/ja` font folder.
- Fixed final placement labels for Swiss + Top Cut, group stages, and pure elimination events, so players outside Top 8 are no longer shown as unranked.

## 日本語

- 言語を切り替えた後も一部の動的 UI 文言が中国語のまま残る問題を修正しました。
- PDF レポートは選択した対象言語の文言で生成され、`/data/fonts/zh`、`/data/fonts/en`、`/data/fonts/ja` の対応する言語フォルダを優先して使用します。
- スイスドロー + Top Cut、グループステージ、純粋なトーナメント形式の最終順位表示を修正し、Top 8 以外の選手が未順位として表示されないようにしました。
